### [Main Concept 1]

**What it is:**
A variable is basically a named storage for data, is like a box

**Why it exists:**
It basically used whenever you want to save data and give them a name

**How it works:**
When you delcare a variable, you use let or const, let is for variables that can change its value once you assign something,
For example: let message; message="Hello"; Here we are declaring that the variable message will have the value of "Hello".
You will also have Const, meaning constant, basically a type of variable that you can only assign the value once, if you 
try to redeclare the value of a previous constant, it will not let you execute. 
Whenever you declare a variable you basically save the data in a space in the memory, like a box, and the key to access this
box is the name of the variable.
