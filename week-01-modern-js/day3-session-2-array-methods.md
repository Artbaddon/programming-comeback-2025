### Array methods

**What it is:**
Array methods are methods that the array object provides to make things easier, they are split into groups:

Add/remove:
Methods to add or remove items.

Splice: 
It can insert, remove and replace elemements

Slice:
returns a new copy from an array

Concat:
Returns a new copy from an array with something added

ForEach:
this 

**Why it exists:**
Array methods make the managment of arrays easier

**How it works:**
they are a lot of them the most important ones are:

push: Adds item to the end.
Pop: removes an item of the end.
shift: removes an item from the beginning
unshift: adds item to the beginning
splice: it can delete, insert and replace, it need some args: the start position, the number of deletions and the elements to add.
slice: it returns a copy from an array of the elements between a range, start and end
concat: creates an copy from an array and additional items
