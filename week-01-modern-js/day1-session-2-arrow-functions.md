### Arrow functions

**What it is:**
Arrow functions are basically an simplier and concise way to create functions, is something more synctactical and is often better than functions but have some differences.

They dont have this, means that when you use this inside of them, it takes it from outside

They cant be called with new, so they cant be constructors

They do not have arguments, meaning that they cant access the propetrty args of a normal functions
**Why it exists:**
Its often used for simplying the way we write functions and is a way to shorten the process of creating and using functions. is often used for writing small functions.

**How it works:**
There are some ways of how you can use them, for example you can create a 1 line function that takes 2 args and if the return is only 1 line, it can be implicit like let var = (a ,b) => a + b, in this case the return is not necessary because it is only one line.

they can also be multiline in which the only change will be that you will have to put {} and a return value


**When to use:**
They are often used when you want to create functions that dont leave the current context
